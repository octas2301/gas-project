# 広告運用 レ点（判定列）の要件整理

分析実行後、各シートの **1列目「判定」**（レ点＝チェックボックス）を **どの条件で ON/OFF にするか** を整理しました。

---

## 方針（④ 推奨だけレ点ON）

- **「除外」「昇格」「調整」および楽天の入札強化推奨** の行だけ **レ点ON** とする。
- 上記に当てはまらない行はシートに出力しない（＝候補シートに出る行はすべて推奨＝レ点ONで統一）。

---

## シート別・レ点の扱い

| シート名 | 出力される行の条件 | レ点（判定列） | 備考 |
|----------|--------------------|----------------|------|
| **除外候補（Amazon）** | 判定結果が **「除外」** の行だけ | **すべて ON** | ROAS が AMZ_ROAS_KEEP 未満のキーワード。もともと「除外」のみ出力のため、全行レ点ON。 |
| **入札調整候補（Amazon）** | 判定結果が **「昇格」** または **「調整」** の行だけ | **すべて ON** | オート→マニュアル移動候補・入札アップ/ダウン候補。もともと昇格/調整のみ出力のため、全行レ点ON。 |
| **除外候補（楽天）** | 推奨アクションが **「除外」** の行だけ | **すべて ON** | ROAS が RAK_ROAS_LIMIT 未満などで「除外」判定。もともと除外のみ出力のため、全行レ点ON。 |
| **入札強化候補（楽天）** | ROAS が **好調（≧ RAK_ROAS_GOOD）** の行だけ | **すべて ON** | 以前は好調行をレ点OFFにしていたが、④で「推奨だけレ点ON」に合わせて **レ点ON** に変更済み。 |

---

## Amazon と 楽天 の対応関係

| 種類 | Amazon | 楽天 |
|------|--------|------|
| **除外** | 除外候補（Amazon）に **「除外」** の行だけ出力 → **レ点ON** | 除外候補（楽天）に **「除外」** の行だけ出力 → **レ点ON** |
| **入札** | 入札調整候補（Amazon）に **「昇格」「調整」** の行だけ出力 → **レ点ON** | 入札強化候補（楽天）に **好調** の行だけ出力 → **レ点ON** |

- **Amazon** … 除外用・入札用で **別シート**（除外候補 / 入札調整候補）。それぞれ「その判定の行だけ」を出しており、出ている行はすべて推奨のため **すべてレ点ON**。コード上も `outRow = [true, ...]` のまま変更なし。
- **楽天** … 除外用・入札用で **別シート**（除外候補 / 入札強化候補）。除外は全行レ点ON。入札強化は **好調行だけ** を出力しており、その行を ④ で **レ点OFF → レ点ON** に変更済み。

---

## コード上の対応箇所

| 対象 | コード上の動き |
|------|----------------|
| Amazon 除外 | `actionType === "除外"` のとき `amzNegData.push(outRow)`。`outRow[0] = true` のためレ点ON。 |
| Amazon 入札 | `actionType === "昇格" or "調整"` のとき `amzPosData.push(outRow)`。`outRow[0] = true` のためレ点ON。 |
| 楽天 除外 | `res.action !== "維持"` のとき `rakNegData.push(outRow)`。`outRow[0] = true` のためレ点ON。 |
| 楽天 入札 | `roasCheck >= RAK_ROAS_GOOD` のとき `rakPosData.push(p)`。`p[0] = true` に設定（④で変更）。 |

---

## まとめ

- **除外** … Amazon・楽天とも「除外候補」シートに **除外判定の行だけ** 出力し、**すべてレ点ON**。
- **入札** … Amazon は「入札調整候補」に **昇格・調整の行だけ** 出力して **すべてレ点ON**。楽天は「入札強化候補」に **好調の行だけ** 出力し、④の対応で **すべてレ点ON** に統一済み。
- いずれも「候補として出ている行＝推奨」なので、**レ点はすべてON** で揃えている。
