# 外部スキャン 本番連携 仕様と実装

## 概要

- **案A（採用）**: メニューを開くとスキャン用 URL（外部スキャンページ）に遷移する。手入力時のみ「手入力」ボタンで GAS のフォームへ。
- **スキャン後の戻り**: 読み取り成功時に自動で GAS に戻る。テスト時（`?test=1` または `?manualReturn=1`）は「GAS に戻る」ボタンで手動戻り。
- **別タブで開く**: カメラ許可のため、外部スキャンは **必ず別タブ（target="_blank" / window.open）** で開く。同一タブで開くと `isTopWindow: false` となり、多くの環境で Permission denied になる。

## 実装内容

### 1. スキャン優先メニュー

| メニュー | 動作 |
|----------|------|
| 出荷・移動・入荷・商品 | タップで外部スキャンページを **別タブ** で開く。ページ表示時にスキャン自動開始。 |
| 棚卸 | タップで GAS の棚卸画面。「スタート」押下後に外部スキャンページを **別タブ** で開く。 |
| 差異 | タップで GAS の差異一覧を表示（スキャン優先にしない）。**差異タブ内のみ**「棚卸確定後の修正でスキャン」リンクを設置し、押下で外部スキャンを別タブで開く。 |
| 在庫・設定 | 従来どおり GAS 内でタブ切り替え。 |

### 2. 外部スキャン URL

- **設定場所**: GAS の Script Properties の `EXTERNAL_SCAN_BASE_URL`。未設定時は `https://octas2301.github.io/gas-project/inventory-app/external-scan-poc.html` を使用。
- **変更時**: プロジェクトの「プロジェクトの設定」→「スクリプト プロパティ」で `EXTERNAL_SCAN_BASE_URL` を追加・編集。

### 3. 外部スキャンページ（external-scan-poc.html）

- **スキャン成功時**: `manualReturn` / `test` が無い場合は即座に `returnUrl + jan` へ遷移（自動で GAS に戻る）。
- **テスト時**: URL に `?test=1` または `?manualReturn=1` を付けると自動遷移せず「GAS に戻る」ボタンを表示。
- **手入力**: 「手入力（GASで入力）」リンクで、`view` パラメータに応じた GAS 画面（jan なし）へ遷移。
- **ページ表示時**: returnUrl または gasUrl が設定されていて、かつ test/manualReturn でない場合は約 400ms 後にスキャン自動開始。

### 4. GAS 側（Code.gs / index.html）

- **doGet**: `view`, `jan`, `adjust` を受け取り、テンプレートに渡す。
- **戻り先**: `view=shipping&jan=`, `view=count&adjust=1&jan=` など。外部ページは `returnUrl` にこれらを渡され、読み取り後に `returnUrl + encodeURIComponent(jan)` でリダイレクト。
- **index.html 初期表示**: `jan` があれば該当 view の JAN 入力欄にセットし blur で lookup。`adjust=1` かつ view=count の場合は棚卸確定後修正セクションを開き、`adjust-jan` にセット。

### 5. 変更ファイル一覧

| ファイル | 変更内容 |
|----------|----------|
| Code.gs | externalScanBaseUrl, gasExecUrl, returnUrl* をテンプレートに渡す。jan, adjust を doGet で受け取りテンプレートに渡す。 |
| index.html | 出荷・移動・入荷・商品のナビを外部スキャン URL に変更（target="_blank"）。差異は GAS の差異一覧表示にし、差異タブ内にのみ「棚卸確定後の修正でスキャン」リンクを設置。棚卸はスタート成功時に window.open(scanUrl, '_blank') で外部スキャンへ。data-scan-url の & は HTML で &amp; に。URL の jan/adjust で入力欄セット・adjust セクション表示。 |
| external-scan-poc.html | スキャン成功時の自動遷移。test/manualReturn 時は「GAS に戻る」表示。手入力リンク追加。ページ表示時のスキャン自動開始。 |

## テスト

- **本番動作**: 出荷タブをタップ → **別タブで** 外部スキャンが開きカメラ起動 → バーコード読み取り → 自動で GAS 出荷画面に戻り JAN が入る。
- **カメラが起動しない場合**: 環境情報の **isTopWindow** を確認。false のときは同一タブや iframe 内で開いているため、リンクを **別タブで開く**（target="_blank"）になっているか・GAS の再デプロイを確認する。
- **手動戻り（テスト）**: 外部スキャンの URL に `?test=1` を付けて開く → 読み取り後も自動遷移せず「GAS に戻る」で戻る。
- **棚卸**: 棚卸タブ → スタート → **別タブで** 外部スキャンへ遷移 → 読み取り → 自動で GAS 棚卸に戻る。
- **差異**: 差異タブで一覧を表示。「棚卸確定後の修正でスキャン」をタップすると別タブで外部スキャンが開き、戻り先は棚卸の確定後修正セクション。
