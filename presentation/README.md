# 車中泊×シュノーケル 10分プレゼン資料

構成C（大家族最適解）× A（王道ストーリー）に基づく社内プレゼン用素材です。

## ファイル一覧

| ファイル | 説明 |
|----------|------|
| `車中泊×シュノーケル_画像プレースホルダー版.pptx` | **本編（19枚・画像挿入位置付き）**。左に画像ゾーン・右にテキストのレイアウトで、プレースホルダーに「何の画像を入れるか」のブリーフを記載。 |
| `車中泊×シュノーケル_装飾版.pptx` | **装飾適用版**。南の海テーマ（ネイビー・ターコイズ・コーラル）でトンマナ統一・表スタイル・アクセント線を反映した版。画像差し込み後に `apply_southern_sea_decoration.py` を実行して生成。 |
| `画像ブリーフ一覧.md` | スライド番号・配置位置・検索キーワード・縦横比・雰囲気の一覧。画像探し・差し替え用。 |
| `create_snorkel_slides.py` | 本編 .pptx を生成する Python スクリプト |
| `apply_southern_sea_decoration.py` | 既存 .pptx に南の海テーマの装飾を上乗せするスクリプト（表スタイル・アクセント線・テキスト色）。 |
| `create_south_sea_with_images.py` | **提供いただいた4枚の南の海画像を埋め込み**、テンプレ風（円形タイトル・左画像＋右テキスト・THANK YOU）で一から生成。 |
| `車中泊×シュノーケル_南の海スタイル.pptx` | 上記スクリプトで生成。4枚の画像をタイトル・スポット紹介・まとめ・THANK YOU に使用。 |
| `images/` | 南の海の背景画像を置くフォルダ。`create_south_sea_with_images.py` はここから4枚を自動検出して使用。 |
| **html_slides/** | **【おすすめ】南の海テンプレをイメージ通りに再現**。Zenn 記事方式で HTML スライド → ブラウザで PDF 保存 → pptx 変換。`index.html` を開き、印刷で PDF に保存してから Adobe Acrobat 等で pptx に変換。 |
| `Gamma用プロンプト.md` | Gamma に貼ってスライドを自動生成するためのプロンプト |
| `10分読み上げ原稿.md` | 本番用の読み上げ原稿（約10分） |
| `画像挿入指示.md` | スライド別の写真・デザイン指示（概要版） |

## PowerPoint を再生成する場合

```bash
cd presentation
python -m pip install -r requirements.txt
python create_snorkel_slides.py
```

生成された `車中泊×シュノーケル_画像プレースホルダー版.pptx` を開き、画像を挿入してください。

**装飾を適用する場合**（画像差し込み後）：

```bash
cd presentation
python apply_southern_sea_decoration.py
```

入力: `車中泊×シュノーケル_画像プレースホルダー版.pptx`  
出力: `車中泊×シュノーケル_装飾版.pptx`（南の海テーマ・表スタイル・アクセント線・トンマナ統一）

Googleスライドで使う場合は、ファイルをドライブにアップロードし「Googleスライドで開く」で変換できます。
